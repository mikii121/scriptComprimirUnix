# scriptComprimirUnix
Script para comprimir y descomprimir archivos en unix

```
Programa que comprime el archivo/carpeta que le pases como argumento.

Usage:
	./tree.sh [PARAMS]

Options:
	[-h|--help]  			Muestra el siguiente mensaje de ayuda.
	[-d|--dir directorio]  	Directorio que queremos comprimir.
	[-c|--compresion tipo]  Tipo de compresión que usaremos.
							tar.gz | zip | rar | tar | gz.
	[-f|--file archivo]		Archivo que vamos a comprimir.
	[-u|--uncompress]		Descomprimir un archivo comprimido.
							Debes utilizar la opción -f para elegir el archivo.

```
